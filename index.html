Apprendre à créer un site web facilement

En découvrant les meilleurs outils pour apprendre le plus radpidement possible.

Bonjour et bienvenue sur ce site web qui est mon produit final pour le projet personnel (si jamais vous ne savez qu'est ce qu'est le projet personnel, cliquez simplement sur le symbole de l'ib situé en haut à gauche du site web). Je m'appelle Sebastian et j'ai appris les langages de code HTML et CSS dans les 5 derniers mois afin de créer la page web sur laquelle vous être en train de lire ce texte, et j'ai fait cela pour plusieurs raisons:

• Tout d'abord, c'est le domaine d'apprentissage que j'aimerai poursuivre rendu à l'université, ainsi que celui dans lequel je souhaite travailler, et donc c'est une bonne façon de prendre de l'avance dans mes apprentissages, et c'est aussi une bonne façon de mettre à mettre en pratique ce j'apprends, ainsi que de savoir si j'ai bien compris ce que j'ai appris.

• Ensuite, j'ai décidé de créer un site web sur comment apprendre à créer un site web puisque je trouve que c'est très dur à trouver des bonnes sources, puisqu'il y a tellement de sources sur internet, mais aussi car il y a differentes façons d'apprendre. Par exemple, il y a des gens qui trouve que just un site web avec toutes les éléments/commandes et une explication de chacun d'entres eux est suffisant pour qu'il comprenent tout. D'autre part, il y a des gens qui apprenent beaucoup mieux visuellement, donc principalement en regardant des vidéos sur le sujet même.

• Finalement, j'ai décidé qu'un site web informatique était le meilleur type de site web à faire. Malgré le fait que je connais assez bien les langages de code HTML et CSS, je n'ai pas pu apprendre le langage qui a comme role de rendre le site web interactif (JavaScript), alors je n'avais pas autant d'options lorsque je choisissais le type de site web que j'aillais créer.

Que vais-je apprendre sur ce site web?
