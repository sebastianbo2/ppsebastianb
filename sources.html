La raison pour laquelle je ne parle pas trop de sites web dans me sources, et pourquoi je ne les recommende pas est parce que c'est habituellement moins interactif, dans le sens qu'il faut habituellement cliquer sur pleins de liens afin d'arriver à la prochaine leçon, et lorsqu'on suit un tutoriel sur YouTube. Source 4 : La façon gratuite de mettre un site web sur Internet -- Voire vidéo ici

Voici la source qui m'a permis de créer une autre version de mon site web gratuitement, à partir d'une application nommée GitHub. Cette application permet de créer plusieurs dépots/groupe de fichiers dans lequel on inclus nos projets et nos sites web. Ce n'est pas la façon professionelle de créer des sites web, puisque le lien aura le nom de la plateform (Github), mais c'est une très bonne ressource pour mettre un site web en ligne. Une des choses les plus importantes dont il a parlé est la création de la page principale : Lorsqu'on crée un site web, notre page principale doit toujours avoir le nom de "index.html" cette page est celle qui sera ouverte lorsqu'on cherche à partir d'un lien (ex: www.ppsebastianb.rowebca.ca/index.html = www.ppsebastianb.rowebca.ca : les deux liens ouvrent la même page). Pour le reste de la vidéo, il parle du domaine web, de comment les obtenirs et de comment se trouver un bon hébergeur pour notre site web : Un hébergeur est une companie qui sert comme médium à travers lequel on peut partager notre site web (ex: Rowebca). C'est util d'avoir un hébergeur puisqu'il offre de la protection pour notre sites web, ainsi que rend le site web plus rapide et plus performant.

Cette vidéo m'a beaucoup appris sur la construction d'un site web, en parlant de tous les sujets à aborder dans une page web, de certaines "règles" de décoration, ainsi que quelques exemples de sites web. Cette vidéo m'a aussi inspiré à aller voire beaucoup de pages web afin de trouver des aspects qui sont commun entre eux, et les inclures dans ma page web, puisque ceci est ma première page web, et je n'ai pas d'expérience avec cela.
